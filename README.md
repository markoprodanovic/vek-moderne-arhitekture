# Vek Moderne Arhitekture

A website dedicated to the book "VEK MODERNE ARHITEKTURE" — featuring curated reference links, author info, an interactive map, and a contact form.

Live at [vek-moderne-arhitekture.info](https://vek-moderne-arhitekture.info)

## Development

```bash
pnpm dev    # Start dev server at localhost:3000
pnpm build  # Production build
pnpm lint   # Run ESLint
```

## Stack

Next.js · React · TypeScript · Tailwind CSS · Vercel
